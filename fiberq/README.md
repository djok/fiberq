# FiberQ (QGIS plugin)

Tools for planning and designing GPON/FTTH networks in QGIS: BOM export (CSV/XLSX), optical slack/reserves, cable branching (branch_index + offset), Publish to PostGIS (Append/Upsert/Overwrite + ANALYZE), fiber break measurement tool.

Website: https://www.fiberq.net/
Contact: vukovicvl@fiberq.net

## Installation
1. Download the ZIP and extract the folder `FiberQ` into your QGIS profile:
   `.../QGIS/QGIS3/profiles/default/python/plugins/`
2. Enable the plugin in **Plugins → Manage and Install Plugins…**


## Requirements
- QGIS ≥ 3.22
- (Optional) `xlsxwriter` for XLSX export (otherwise use CSV)
